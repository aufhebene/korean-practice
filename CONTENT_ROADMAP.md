# 컨텐츠 보강 로드맵

## 현재 상태 분석

### 현재 컨텐츠 현황

| 모듈 | 현재 개수 | 목표 | 부족한 점 |
|-----|---------|-----|---------|
| 어휘 | 5문제 (하드코딩) | 500+ | 카테고리 없음, 레벨 구분 없음 |
| 문법 | 5문제 (하드코딩) | 100+ | 조사만 있음, 어미/문형 없음 |
| 회화 | 5개 시나리오 (각 3-4문장) | 30+ | 문장 수 부족, 난이도 단순 |
| 듣기 | 5문제 (하드코딩) | 100+ | 실제 오디오 없음 (TTS만) |

### 주요 부족한 점

1. **컨텐츠 양 절대 부족** - 학습할 내용이 너무 적음
2. **레벨 시스템 없음** - 초급/중급/고급 구분 없음
3. **카테고리 선택 불가** - 원하는 주제 선택 불가
4. **학습 진도 저장 안됨** - 새로고침하면 리셋
5. **퀴즈 유형 단조로움** - 각 모듈에 1가지 퀴즈만
6. **한글(자모) 학습 없음** - 완전 초보자용 컨텐츠 없음
7. **복습 시스템 없음** - 틀린 문제 다시 풀기 없음

### 1.2 문법 확장 (5개 → 100개)

#### 문법 카테고리
```
📝 조사 (20개)
├── 주격 조사: 이/가, 은/는, 께서
├── 목적격 조사: 을/를
├── 부사격 조사: 에, 에서, 에게, 한테, (으)로
├── 관형격 조사: 의
└── 접속 조사: 와/과, (이)랑, 하고

📝 어미 - 종결 (20개)
├── 평서형: -아/어요, -습니다/ㅂ니다, -아/어
├── 의문형: -아/어요?, -습니까?, -니?
├── 명령형: -(으)세요, -아/어라
├── 청유형: -아/어요, -(으)ㅂ시다
└── 감탄형: -네요, -군요

📝 어미 - 연결 (20개)
├── 나열: -고, -(으)며
├── 대조: -지만, -(으)ㄴ데
├── 이유: -아/어서, -(으)니까, -기 때문에
├── 조건: -(으)면, -아/어야
└── 시간: -(으)면서, -자마자, -기 전에

📝 표현/문형 (40개)
├── 가능/불가능: -ㄹ 수 있다/없다
├── 의무: -아/어야 하다
├── 추측: -ㄹ 것 같다, -(으)ㄴ/는 것 같다
├── 희망: -고 싶다
├── 경험: -아/어 본 적이 있다
├── 진행: -고 있다
├── 완료: -아/어 버리다
├── 비교: -보다, 만큼
├── 인용: -다고 하다, -냐고 하다
└── 높임말: 존댓말 체계
```

#### 문법 데이터 구조
```typescript
{
  id: string
  pattern: string            // "-아/어서"
  name: string               // "이유를 나타내는 연결 어미"
  category: "particle" | "ending" | "expression"
  subcategory: string
  level: 1 | 2 | 3
  explanation: {
    en: string
    ko: string
  }
  formation: string          // 만드는 방법
  examples: {
    sentence: string
    translation: string
    breakdown?: string       // 문장 분석
  }[]
  commonMistakes?: string[]  // 자주 틀리는 점
  comparison?: string[]      // 비슷한 문법과 비교
  quiz: {
    type: "fill-blank" | "select" | "correct-error"
    questions: QuizQuestion[]
  }
}
```

## Phase 2: 새로운 학습 모듈 추가

### 2.1 한글 학습 (완전 초보자용)

```
📖 한글 마스터
├── 1. 모음 (10개)
│   ├── 기본 모음: ㅏ ㅓ ㅗ ㅜ ㅡ ㅣ
│   └── 복합 모음: ㅐ ㅔ ㅚ ㅟ 등
├── 2. 자음 (14개)
│   ├── 기본 자음: ㄱ ㄴ ㄷ ㄹ ㅁ ㅂ ㅅ ㅇ ㅈ ㅎ
│   └── 쌍자음: ㄲ ㄸ ㅃ ㅆ ㅉ
├── 3. 받침 (27개)
│   ├── 단일 받침
│   └── 겹받침
├── 4. 글자 조합
│   └── 자음 + 모음 + (받침)
└── 5. 발음 규칙
    ├── 연음
    ├── 경음화
    └── 비음화
```

#### 한글 퀴즈 유형
- **매칭**: 자음/모음 이름과 글자 연결
- **발음 듣기**: 소리 듣고 글자 선택
- **쓰기 연습**: 획순 따라 그리기 (Canvas API)
- **조합하기**: 자음+모음 조합해서 글자 만들기

## Phase 3: 퀴즈 유형 다양화

### 현재 퀴즈 유형 (모듈별 1개)
| 모듈 | 현재 유형 |
|-----|---------|
| 어휘 | 의미 선택 |
| 문법 | 빈칸 채우기 |
| 회화 | 단어 조합 |
| 듣기 | 문장 조합 |

### 추가할 퀴즈 유형

#### 어휘
1. **플래시카드** - 카드 뒤집기로 암기
2. **단어 → 의미** - 한국어 보고 영어 선택 ✅ (현재)
3. **의미 → 단어** - 영어 보고 한국어 선택
4. **빈칸 채우기** - 문장 속 단어 맞추기
5. **발음 듣기** - 발음 듣고 단어 선택
6. **철자 맞추기** - 글자 조합해서 단어 만들기
7. **그림 보고 선택** - 이미지와 단어 매칭

#### 문법
1. **조사 선택** ✅ (현재)
2. **어미 선택** - 문장에 맞는 어미 선택
3. **문장 완성** - 문법에 맞게 문장 완성
4. **오류 찾기** - 틀린 부분 찾아 고치기
5. **문장 변환** - 반말 ↔ 존댓말 변환
6. **순서 배열** - 단어 순서 맞추기

#### 회화
1. **단어 조합** ✅ (현재)
2. **대화 완성** - 상황에 맞는 응답 선택
3. **롤플레이** - AI와 대화 (음성 인식)
4. **상황 판단** - 적절한 표현 선택
5. **문화 퀴즈** - 상황에 맞는 행동 선택

#### 듣기
1. **문장 조합** ✅ (현재)
2. **받아쓰기** - 직접 타이핑
3. **의미 파악** - 들은 내용의 의미 선택
4. **속도 조절** - 0.5x, 1x, 1.5x 속도
5. **대화 듣기** - 긴 대화 듣고 문제 풀기

## 우선순위 정리

### 즉시 (1주일 내)
1. ✅ 어휘 데이터 50개 → 100개 확장
2. ✅ 문법 데이터 5개 → 20개 확장
3. ✅ 회화 시나리오 문장 수 증가

### 단기 (1개월 내)
1. 카테고리 선택 기능
2. 레벨 선택 기능
3. 퀴즈 유형 2-3개 추가
4. 로컬 스토리지로 진도 저장

### 중기 (3개월 내)
1. 한글 학습 모듈
2. 사용자 인증 (데이터 영구 저장)
3. 연속 학습/레벨 시스템
4. 학습 통계 대시보드

### 장기 (6개월 이상)
1. AI 기반 맞춤 학습
2. 음성 인식 말하기 연습
3. 커뮤니티 기능
4. 프리미엄 컨텐츠

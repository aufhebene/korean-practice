# 한국어 공부 웹 애플리케이션 기획서

## 1. 프로젝트 개요

### 1.1 프로젝트명
**KoreanPractice** - 게임화 기반 인터랙티브 한국어 학습 웹 애플리케이션

### 1.2 컨셉
- **게임화(Gamification)** 요소를 강조한 한국어 학습 앱
- 귀여운 마스코트 캐릭터를 활용한 친근한 학습 경험
- 인터랙티브 퀴즈 중심의 능동적 학습 방식
- 데이터 기반 반복 학습 시스템

### 1.3 디자인 컨셉
- 파스텔 톤의 깔끔하고 부드러운 배경
- 마스코트 캐릭터 (예: 호랑이, 토끼, 또는 한국 전통 캐릭터)
- 보라색/파란색 계열의 메인 컬러
- 직관적이고 미니멀한 UI

### 1.4 대상 사용자
- 한국어를 배우고자 하는 외국인
- 초급부터 고급까지 다양한 레벨의 학습자
- K-POP, 드라마 등 한국 문화에 관심 있는 학습자

## 2. 학습 모듈 구조 (수정됨)

### 2.1 핵심 원칙
- **각 학습 모듈은 독립적으로 운영**
- 사용자가 원하는 모듈을 선택하여 학습
- 모듈별로 진도, 통계, 복습 시스템 별도 관리

### 2.2 학습 모듈 목록

| 모듈 | 설명 | 퀴즈 유형 |
|-----|------|----------|
| **어휘** | 단어 암기 및 의미 학습 | 빈칸 채우기, 의미 선택, 플래시카드 |
| **문법** | 문법 패턴 및 조사 학습 | 문장 완성, 올바른 문법 선택 |
| **회화** | 상황별 대화 표현 학습 | 단어 조합, 대화 완성 |
| **듣기** | 발음 및 청취 학습 | 리스닝 퀴즈, 받아쓰기 |

### 2.3 학습 플로우 (수정됨)

```
[홈 화면]
    │
    ├── [어휘 학습] ──→ 어휘 전용 퀴즈 세션
    │       └── 새 단어 / 복습 / 재도전
    │
    ├── [문법 학습] ──→ 문법 전용 퀴즈 세션
    │       └── 새 문법 / 복습 / 재도전
    │
    ├── [회화 학습] ──→ 시나리오 선택 → 회화 퀴즈
    │       └── 카페 주문 / 쇼핑 / 비즈니스 등
    │
    └── [듣기 학습] ──→ 리스닝 전용 퀴즈 세션
            └── 문장 듣기 / 대화 듣기
```

## 5. 데이터 모델 (수정됨)

### StudyModule (학습 모듈)
```typescript
type StudyModule = "vocabulary" | "grammar" | "conversation" | "listening";
```

### ModuleProgress (모듈별 진도)
```typescript
{
  id: string
  userId: string
  module: StudyModule
  newCount: number        // 새 항목 수
  reviewCount: number     // 복습 항목 수
  retryCount: number      // 재도전 항목 수
  totalMastered: number   // 마스터한 항목 수
  lastStudiedAt: Date
}
```

### VocabularyItem (어휘)
```typescript
{
  id: string
  korean: string
  pronunciation: string
  meaning: string
  category: string
  level: number
  examples: string[]
}
```

### GrammarItem (문법)
```typescript
{
  id: string
  pattern: string         // "~는/은"
  explanation: string     // "주어를 나타내는 조사"
  examples: string[]      // ["나는 학생입니다", "책은 여기 있어요"]
  level: number
}
```

### ConversationScenario (회화 시나리오)
```typescript
{
  id: string
  title: string           // "카페에서 주문하기"
  category: string        // "일상생활"
  level: number
  sentences: {
    korean: string
    english: string
    words: string[]       // 단어 조합용
  }[]
}
```

### ListeningItem (듣기)
```typescript
{
  id: string
  audioUrl: string
  transcript: string      // 정답 텍스트
  words: string[]         // 선택지 단어들
  level: number
}
```

## 6. 퀴즈 유형 정리 (모듈별)

### 어휘 학습
| 퀴즈 유형 | 설명 |
|----------|------|
| 의미 선택 | 한국어 단어 보고 영어 의미 선택 |
| 단어 선택 | 영어 의미 보고 한국어 단어 선택 |
| 빈칸 채우기 | 예문 속 빈칸에 단어 채우기 |

### 문법 학습
| 퀴즈 유형 | 설명 |
|----------|------|
| 조사 선택 | 문장에 알맞은 조사 선택 |
| 어미 선택 | 문장에 알맞은 어미 선택 |
| 문장 완성 | 올바른 문법 패턴으로 문장 완성 |

### 회화 학습
| 퀴즈 유형 | 설명 |
|----------|------|
| 단어 조합 | 단어 카드를 순서대로 배치해 문장 완성 |
| 대화 완성 | 대화 상황에 맞는 응답 선택 |

### 듣기 학습
| 퀴즈 유형 | 설명 |
|----------|------|
| 문장 조합 | 오디오 듣고 단어 순서 맞추기 |
| 받아쓰기 | 오디오 듣고 직접 입력 |
